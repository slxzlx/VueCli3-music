# vuecli3-music
因为获取不了qq音乐声音的api 目前只插入一首歌选择所有的歌曲只播放这一首。
使用
***
使用了vue-awsome-swiper作为轮播图插件，better-scroll作为滑动插件。使用了localstorage来存储搜索历史。
歌手列表的数据使用的mock假数据，其他的均为使用devserver跨域访问的qq音乐的api获取数据
